# ERO1

# Instrctions d’installation et d’exécution

Installer les modules python necessaires puis lancer les fichiers executables.

# Structure du rendu

Un dossier General contenant des informations sur les quartiers de Montreal ainsi que sur des graphes generes aleatoirement.
Un dossier Data contenant les donnees geologique des quartiers des 5 quartiers de Montreal.
Un dossier Miscellanous contenant des tests sur differents algorithmes de generation et de parcours de graphes.
Un dossier Snowplow contenant le code necessaire au parcours des deneigeuses.
Un dossier Drone contenant le code necessaire au parcours du drone.
Des fichiers PNG et des animations mp4 sont repartis un peu partout dans le projet et permettent une comprehension plus aboutit sur le projet.
